### Pre-Release Notes

```diff
- WARNING: Multiline is not yet supported
```
- RPN-ish Arithmetic Added
```peridot
# Peri.Py
(1 + 2) * 3 # ((1 + 2) * 3)
1 + 2 * 3   # (1 + (2 * 3))

# Rusty Peri.dot
1 2+ 3*     # ((1 + 2) * 3)
1 (2 3*)+   # (1 + (2 * 3))
```
